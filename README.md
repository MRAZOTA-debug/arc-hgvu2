# Reply → Quote Plugin

Плагин для exteraGram, который заменяет текст "Ответ" на "Цитировать" в интерфейсе Telegram.

## Установка

1. Скачайте файл `reply_rename.py`
2. Откройте exteraGram
3. Перейдите в Настройки → Плагины
4. Нажмите "Установить плагин"
5. Выберите скачанный файл
6. Включите плагин в списке

## Использование

После установки плагин работает автоматически. Текст "Ответ" в интерфейсе будет заменен на "Цитировать".

## Требования

- exteraGram >= 12.5.1
- SDK >= 1.4.3.6
